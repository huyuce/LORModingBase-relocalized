#运行前注意！！！
1，使用请订阅创意工坊BaseMod（https://steamcommunity.com/sharedfiles/filedetails/?id=2603522001）并且勾选此mod至少载入游戏主界面一次。
2，此时您的游戏目录中将会出现\Library Of Ruina\LibraryOfRuina_Data\Managed\BaseMod
3，复制\Library Of Ruina\LibraryOfRuina_Data\Managed\BaseMod\Localize中的“cn”文件夹并且改名为“kr”
4，复制\Library Of Ruina\LibraryOfRuina_Data\Managed\BaseMod\Story中的“cn”文件夹并且改名为“kr”
（原代码需求默认本地化为kr，但创意工坊内BaseMod有现成cn本地化于是直接套用）
启动程序，选择正确的目录。
